## 🚀 Setup

```bash
# Clone the repository
git clone https://github.com/manoel-lopes/todo-list-app.git

# Access the project folder
cd todo-list-app

# Install the dependencies
npm install

# Run the application
npm run dev

# The server will start on port:3000 - go to http://localhost:3000
```
